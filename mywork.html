Misc. Projects

Partie 3D Application
[WORK IN PROGRESS FOR THIS WEBSITE] My main task for my summer 2022 internship working for Partie was to create a Windows application version of Partie's website using my present Unity knowledge. I took it a step further and made it in 3D as both a challenge to myself, and knowing that Unity's 3D space can easily be ported into VR, allowing for hypothetical Metaverse capabilities. I had to do this by using their API and creating Unity C# Post/Get requests.

Regardless, with the time I had, this application currently displays a "feed" from their website, and you can use horizontal movement keys to move the character through the feed. It will infinitely load and will always load the most updated posts.

You can view the prototype application via WebGL here.

I created a website using the GIPHY API for a major school assignment. You can search anything you'd like to see GIFs of, like you would for a Google image search, and it will display a user-specified number of results, in order.

Color of Darkness
This is a game where the objective is simple: survive. All you have to do is move around in this 2d auto-attacking game. The game will get progressively more difficult, but you will become more powerful as well. Upgrade different weapons and use powerups to fight the constant horde of enemies coming your way!

Cloud Chow
Play as a dragon in this infinite 2D side-scroller, trying to eat as many clouds as you can before you inevitably lose! The game will slowly ramp up in diffuclty, as the speed increases and so does the spawn rate/damage of the storm clouds! Earn the highest score you can!

You can play the game via WebGL here.
Timeless Torture
You wake up in the wrong time period. Find the haunted items around the house, burn them in the fireplace to cleanse the evil from them. After all items are found, a portal will appear, which you have to find to travel to the next time period. You must do all of this before time runs out, and keep going until you reach the present day.

You can find a link to the itch.io page here.
Flappy Flipper
This is a 2D side-scroller phone game where the objective is to get the highest score you can by flying through the icicles with the penguin! Penguins can't actually fly, of course, which is why it has a jetpack strapped to its back!

Class of 21XX
“Class of 21XX” is a puzzle/skill 2D platformer that follows four friends about to graduate from Faith Academy, a school for future space cadets. While they’re well prepared for their final assignment, they aren’t ready to say goodbye to each other. However, an unexpected alien attack puts their test on hold, forcing them to save their school, fix the ship, and go out with one last hurrah.
