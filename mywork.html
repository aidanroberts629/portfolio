Misc. Projects

Partie 3D Application
My main task for my summer 2022 internship working for Partie was to create a Windows application version of Partie's website using my present Unity knowledge. I took it a step further and made it in 3D as both a challenge to myself, and knowing that Unity's 3D space can easily be ported into VR, allowing for hypothetical Metaverse capabilities. I had to do this by using their API and creating Unity C# Post/Get requests. Regardless, with the time I had, this application currently displays a "feed" (infinitely and always updated) from their website.

You can download the prototype application [Desktop Only] here.

WatchList Website
I made this website using the MovieDB API for a major project in a rich media class. There are different filters and search options to look up any movie or television show. You can also favorite search results to effectively make your own "WatchList" (hence the name), and both favorites and search terms are stored in local storage. The website was created using HTML/CSS and JavaScript.

You can view the rich media application [Desktop only] here.

The game will get progressively more difficult, but you will become more powerful as well. Upgrade different weapons and use powerups to fight the constant horde of enemies coming your way!

Cloud Chow
Play as a dragon in this infinite 2D side-scroller, trying to eat as many clouds as you can before you inevitably lose! The game will slowly ramp up in diffuclty, as the speed increases and so does the spawn rate/damage of the storm clouds! Earn the highest score you can!

You can play the game via WebGL [Desktop Only] here.
Timeless Torture
You wake up in the wrong time period. Find the haunted items around the house, burn them in the fireplace to cleanse the evil from them. After all items are found, a portal will appear, which you have to find to travel to the next time period. You must do all of this before time runs out, and keep going until you reach the present day.

You can find a link to the itch.io page [Desktop Only] here.

“Class of 21XX” is a puzzle/skill 2D platformer that follows four friends about to graduate from Faith Academy, a school for future space cadets. While they’re well prepared for their final assignment, they aren’t ready to say goodbye to each other. However, an unexpected alien attack puts their test on hold, forcing them to save their school, fix the ship, and go out with one last hurrah.
